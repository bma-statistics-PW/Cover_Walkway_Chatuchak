## ขอบเขตงาน (BOQ — 17 รายการ)

### หมวด 1 งานโครงสร้าง
- เสาเข็ม ฐานราก โครงสร้างเสา TYPE1 & 1.2 (24 แห่ง)
- เสาเข็ม ฐานราก โครงสร้างเสา TYPE2 (23 แห่ง)
- เสาเข็ม ฐานราก โครงสร้างเสา TYPE3 (5 แห่ง)
- เสาเข็ม ฐานราก โครงสร้างเสา TYPE4 (5 แห่ง)
- หลังคาและคาน TYPE1 & 1.2 / TYPE2 / TYPE3 / TYPE4

### หมวด 2 งานตกแต่ง
- ป้ายโครงการ 3 จุด, ป้ายบอกทิศทาง 4 จุด
- ผนังก่ออิฐมวลเบา ฉาบทรายล้าง 11 แห่ง

### หมวด 3 งานคืนสภาพ
- รื้อย้ายระบบสาธารณูปโภค, ต้นไม้พุ่ม, ต้นไม้ยืนต้น
- ปูกระเบื้องทางเท้า 950 ตร.ม.

### หมวด 4 งานระบบไฟฟ้า
- ระบบไฟฟ้า, โคมฝังฝ้า T8 LED 80 ชุด, โคมกันน้ำ 16 ชุด

## วิธีเปิดใช้งาน Dashboard

### วิธีที่ 1 — GitHub Pages (แนะนำ)
1. ไปที่ **Settings** → **Pages**
2. Source: เลือก **main** branch → **/root**
3. กด **Save** — จะได้ link สำหรับแชร์

### วิธีที่ 2 — ดาวน์โหลดเปิดเอง
1. คลิก **Code** → **Download ZIP**
2. แตกไฟล์ → เปิด `index_updated.html` ด้วย Browser

## ปัญหาอุปสรรคหลัก

1. ข้อจำกัดเวลาเข้าพื้นที่สวนจตุจักร (08:00–15:00 น. เท่านั้น)
2. โครงสร้างสาธารณูปโภคใต้ดินไม่ตรงตามแบบ (TYPE4)
3. ประตูรั้ว MRT ขัดขวางตำแหน่งเสาเข็ม
4. กำลังไฟฟ้าหน้างานไม่เพียงพอ (เชื่อมได้เพียง 2 ตู้)
5. ต้นไม้กีดขวางแนวก่อสร้างหลายจุด
